# pica-dl

> 参考 [lx1169732264/pica_crawler](https://github.com/lx1169732264/pica_crawler)<br>
> 想添加新功能，奈何 Python 早就忘光了，只好重写一个。

哔咔漫画下载器

![演示](./resource/1.gif)

- 交互式命令行
- 排行榜：下载当前排行榜的全部漫画
- 收藏夹：下载当前用户收藏夹的全部漫画
- 关键词搜索：支持多选
- 自动过滤已下载的章节和图片，不会重复下载
- 优先使用环境变量配置，可自动执行，无需在命令行手动选择和输入
